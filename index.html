SLM PRIVATE CHAT

Les règle primordiales

1- Se connecté(e) ou s'inscrire avec un compte Discord anonyme (ne pas savoir qui est derrière l'appareil).
2- Pour ceux qui ont déjà un compte anonyme, pour le bon fonctionnement du chat supprimer votre avatar dans les paramètre de profil de Discord.


3- Respecter la règle 1 et 2.
